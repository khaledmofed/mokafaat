# نظام البطاقات - Cards System

## نظرة عامة

تم إنشاء نظام شامل للبطاقات يسمح للمستخدمين بتصفح وشراء بطاقات من شركات مختلفة مثل STC، شاهد، Hunger Station، وجاذر إن.

## الميزات الرئيسية

### 1. صفحة البطاقات الرئيسية (`/cards`)

- عرض جميع الشركات في شكل كاردات ملونة
- نظام فلترة حسب الفئة (الاتصالات، قنوات تلفزيون، متاجر، ألعاب، إلخ)
- نظام بحث في أسماء الشركات والأوصاف
- نظام بيجينيشن للتنقل بين الصفحات
- تصميم مطابق لصفحة Portfolio

### 2. صفحة تفاصيل الشركة (`/cards/[companyId]`)

- عرض معلومات الشركة مع لوجو ولون مميز
- عرض "العروض المميزة" للشركة
- كل عرض في كارد منفصل مع السعر والمدة والمميزات
- إحصائيات العرض (التقييم، المشاهدات، التحميلات، المفضلة)

### 3. مودال تفاصيل العرض

- عرض تفاصيل كاملة للعرض
- إمكانية تغيير الكمية
- عرض المميزات والتفاصيل
- أزرار الشراء (Apple Pay، شراء سريع)

### 4. صفحات الدفع الإلكتروني (`/cards/[companyId]/payment`)

- اختيار طريقة الدفع (MasterCard، Visa، Apple Pay، Mada)
- خيارات دفع بديلة (المحفظة، نقاط الولاء)
- نموذج إدخال بيانات البطاقة
- صفحة تأكيد الشراء

### 5. صفحة تأكيد الشراء (`/cards/[companyId]/success`)

- رسالة نجاح العملية
- تفاصيل الطلب (رقم الطلب، الشركة، العرض، الكمية، المجموع)
- أزرار تنزيل PDF والعودة للبطاقات

## البيانات الوهمية

تم إنشاء بيانات وهمية شاملة تشمل:

### الشركات

- **STC Pay** (الاتصالات) - لون أرجواني
- **شاهد** (قنوات تلفزيون) - لون أخضر
- **Hunger Station** (توصيل طعام) - لون ذهبي
- **جاذر إن** (ترفيه) - لون بنفسجي

### العروض

كل شركة تحتوي على عروض متعددة مع:

- الأسعار الأصلية والمخفضة
- مدة الصلاحية
- المميزات
- الإحصائيات (التقييم، المشاهدات، إلخ)

## التصميم

- تصميم متجاوب يعمل على جميع الأجهزة
- ألوان مميزة لكل شركة
- تأثيرات hover وانتقالات سلسة
- دعم كامل للغة العربية والإنجليزية
- تصميم متسق مع باقي الموقع

## الروتات

```
/cards - صفحة البطاقات الرئيسية
/cards/[companyId] - صفحة تفاصيل الشركة
/cards/[companyId]/payment - صفحة الدفع
/cards/[companyId]/success - صفحة تأكيد الشراء
```

## الاستخدام

1. المستخدم يضغط على زر "عرض الكل" في قسم البطاقات في الصفحة الرئيسية
2. يتم توجيهه لصفحة `/cards` لعرض جميع الشركات
3. يمكنه الفلترة حسب الفئة أو البحث
4. عند الضغط على شركة، ينتقل لصفحة تفاصيلها
5. يختار العرض المطلوب من العروض المتاحة
6. يفتح مودال التفاصيل لمراجعة العرض وتغيير الكمية
7. يضغط على "شراء سريع" للانتقال لصفحة الدفع
8. يختار طريقة الدفع ويدخل البيانات المطلوبة
9. يؤكد الشراء ويحصل على رسالة نجاح مع تفاصيل الطلب
